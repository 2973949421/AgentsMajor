## 项目使用建议

### 1. Agent Major 内部角色映射

| CS 位置 | Agent Major 职责 |
|---|---|
| IGL | 战术规划 / 回合策略 / 资源分配 |
| AWPer | 高精度关键论点 / 单点突破 / 高风险高收益调用 |
| Entry | 首轮出击 / 激进创意 / 打开局面 |
| Star Rifler | 核心输出 / 关键回合 carry |
| Lurker | 反制 / 偷点 / 找对手逻辑漏洞 |
| Support | 补全细节 / 修复方案 / 提供上下文 |
| Anchor | 防守型论证 / 稳定性校验 |
| Coach | 战术暂停 / 赛前准备 / 赛后复盘 |

### 2. 可信度处理

- **高**：公开资料明确或长期广泛认知。
- **中-高**：公开资料较多，但可能随阵容/地图变化。
- **中**：社区认知或项目推断，适合项目化使用。
- **低-中**：公开资料不足，建议后续继续补强。
- **低**：仅作为占位，需要后续确认。

### 3. 后续可扩展

后续可以为每名选手继续生成：

- Agent 恶搞名；
- 中文外号；
- 技能名；
- 选手卡；
- 击杀播报模板；
- 解说专属称呼；
- 弹幕触发词；
- MVP / EVP 数据权重。
