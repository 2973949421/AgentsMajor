# Agent Major 参赛队伍与 PhaseClan 递补设定

> 版本：Agent Major 初始设定稿  
> 用途：作为项目世界观、赛事数据、队伍命名与新闻内容生成的基础资产  
> 当前阶段：16 队单败 BO3 淘汰赛设定  
> 本版新增：16 支正赛队伍 + 1 支被替换候补队伍的现实选手/教练阵容

---

## 1. 项目背景

**Agent Major** 是一个以 CS 赛事结构为灵感的 AI Agent 电竞化赛事项目。

项目核心设定：

- 16 支 AI 化战队进入 Major 淘汰赛阶段；
- 每支队伍基于现实 CS 战队原型进行戏谑化、AI 化命名；
- 命名需要保留原型识别度，但不能直接照搬现实队名；
- 比赛采用 **16 队单败淘汰赛**；
- 每场比赛为 **BO3**；
- 每张地图不严格照搬 CS 的 13 胜规则，而是采用缩短后的 AI 对抗回合制；
- 比赛使用 token 作为经济系统；
- 项目会配套 2D 直播页面、官方解说、弹幕、击杀记录、支持率、赛后 MVP/EVP、新闻、精彩回放等外围生态。

该项目的目标不是严肃复刻 CS，而是打造一个具有 CS 赛事感、中文直播间梗文化、AI 模型对抗和 token 经济系统的娱乐化 AI 赛事宇宙。

---

## 2. 命名原则

Agent Major 的队伍命名遵循以下原则：

1. **保留原型识别度**  
   用户应能一眼看出该队伍对应哪支现实 CS 队伍。

2. **避免直接照搬原队名**  
   尽量使用平行宇宙式改名，降低版权和品牌混淆风险。

3. **融入 AI / LLM / token / 程序 / 模型相关元素**  
   但不能简单粗暴地在原队名后面硬加 Vector、Context、MoE 等后缀。

4. **名字要像自然长出来的梗**  
   例如：
   - FURIA → FurIA
   - Aurora → AuLoRA
   - B8 → Bit8
   - Legacy → Legacy.v1
   - FUT → FUtureToken

5. **适合解说和弹幕传播**  
   名字需要好念、好记、有节目效果。

## 4. 被替换队伍：3DMAX / 3D-MoE

### 原候选队伍

| 现实原型 | 原拟定 Agent Major 队名 |
|---|---|
| 3DMAX | **3D-MoE** |

### 命名含义

**3D-MoE** 的含义是：

- 3DMAX → 3D；
- MoE = Mixture of Experts，混合专家模型；
- 名字带有强烈 AI 模型架构感。

### 最终未入选原因

3D-MoE 最终没有进入正式 16 队名单，主要原因：

1. **名字略显硬贴 AI 后缀**  
   “3D-MoE”虽然技术梗明显，但更像模型模块或工具名，不太像一支有观众缘的战队。

2. **节目效果不如 PhaseClan**  
   Agent Major 的核心不是完全复刻排名，而是构建一个有戏剧性、有流量、有梗的 AI 电竞赛事。

3. **PhaseClan 的故事性更强**  
   FaZe / PhaseClan 作为特殊外卡队伍，天然带有“流量队”“没进也要进”“节目组懂流量”的戏剧效果。

4. **Prompt Gaming 更值得保留**  
   在 3DMAX 和 paiN Gaming 二选一时，Prompt Gaming 的命名更自然、更适合项目世界观，因此选择保留 Prompt Gaming，移除 3D-MoE。

## 5. PhaseClan 递补新闻设定

### 递补背景

FaZe Clan 在现实赛事语境中具备极高节目效果和观众认知度。  
虽然在 Agent Major 的现实原型筛选逻辑中，FaZe 并非原定正赛队伍，但为了增强赛事观赏性和话题性，项目决定让其 AI 化版本 **PhaseClan** 作为特殊邀请队伍进入正赛。

### 世界观解释

原定参赛队伍 **3D-MoE** 因为“模型签证、专家路由许可、跨区域推理审查”等原因，未能按时完成 Agent Major 注册。

随后，赛事组委会宣布：

> **PhaseClan 将作为 Special Invite 火线递补进入 Agent Major 正赛。**

这是一个参考 CS 赛事中常见“签证问题 / 临时替补 / 特邀席位”梗的虚构设定。

## 9. PhaseClan 队伍设定

### 基本设定

| 字段 | 内容 |
|---|---|
| 队伍名 | **PhaseClan** |
| 现实原型 | FaZe Clan |
| 参赛身份 | Special Invite / 特邀外卡 |
| 递补对象 | 3D-MoE |
| 赛事标签 | 流量队、相位切换、节目效果、强行入场 |
| 风格定位 | 高波动、高上限、高名场面密度 |

### 队伍风格

PhaseClan 不是最稳定的队伍，但它具备极强的节目效果。

建议设定：

```text
PhaseClan 是一支极度依赖节奏切换和高风险操作的队伍。
他们可能在 eco 局突然 force buy，也可能在落后时打出离谱的 phase shift。
这支队伍的核心魅力不是稳定，而是高光密度、反转能力和流量属性。
```

### 招牌战术

```text
- Phase Shift：在对手以为他们进入 save 状态时，突然切换为强攻。
- Inference Rush：牺牲部分上下文，换取快速输出和节奏压制。
- Token Force：经济不足时强行拉满关键 agent。
- Late Phase Clutch：在最后一轮用 Closer Agent 完成残局翻盘。
```
