# 6657 Core

## 目标

这份资产定义 Agent Major 弹幕侧的主风格，不做中和，不做公开柔化支线，直接保留 6657 / 玩机器直播间的核心语言机制。

## 核心机制

- 复读先行：一条句子能不能滚起来，比它是否完整更重要。
- 阴阳反串：表面夸、实则黑，或表面黑、实则抬。
- 结论先行：弹幕先下判决，再看比赛怎么跟上。
- 选手压缩：优先使用单字、黑话、动物化、食物化、符号化别称。
- 研发叙事：离谱操作、强行选择、实验失败都可以进入研发宇宙。
- 群魔乱舞：允许抽象句、错位比喻、低逻辑高传播文本。
- 跨圈共振：MyGO / 木柜子类模板属于核心风格，不是边角补充。
