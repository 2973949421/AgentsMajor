# 文档索引目录

本目录是 Agent Major 的文档导航层。

```text
current-docs.md：当前必读文档和契约入口。
archive-log.md：历史文档迁移记录。
backlog-index.md：长期设想入口。
```

阅读规则：

```text
当前实现判断优先读 current-docs.md。
需要查旧 Phase 或旧 Node/Sector 背景时读 archive-log.md。
需要查长期生态、新闻、奖项、完整赛事时读 backlog-index.md。
```

Archive 和 Backlog 不作为当前执行依据。需要把其中内容重新变成计划，必须先写新的 Phase / N 计划。
