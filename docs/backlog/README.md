## 主题

### 赛事生态

包括：

```text
统计与奖项。
新闻与媒体站。
素材库与队伍 / 选手叙事。
弹幕、解说和转播包装延展。
```

### 完整赛事

包括：

```text
16 队 bracket。
fixture / seed / map pool。
tournament scheduling。
失败恢复。
公开导出 / API 边界。
```

### Web Ops

包括：

```text
队列。
远端部署。
可观测性。
成本控制。
多人访问和权限。
```

## 使用规则

```text
Backlog 可以启发新计划。
Backlog 不能直接覆盖当前 Hex contract。
任何 backlog 项进入实现前，都必须写新的 Phase / N 计划并明确成功标准。
```
